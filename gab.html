HTML Basics

HTML Structure

Every HTML page begins with the <!DOCTYPE html> declaration, which specifies the document type. The main sections include:

  • <html>: The root element that wraps all HTML content.
  • <head>: Contains meta-information, like the page title and link to CSS files.
  • <body>: Holds the visible content of the webpage.
Learn HTML Basics on W3Schools Explore HTML on MDN

Basic HTML Tags

  • <h1> to <h6>: Headers, where <h1> is the largest and <h6> is the smallest.
  • <p>: Paragraphs for regular text.
  • <a>: Anchor tags for hyperlinks. Example: <a href="https://example.com">Link Text</a>
  • <img>: Image tags. Example: <img src="image.jpg" alt="description">
  • <ul>, <ol>, <li>: Unordered lists, ordered lists, and list items.
Try to Code Yourself on W3Schools

Forms

Forms allow user input. Key elements are:

  • <form>: Wraps the entire form content.
  • <input>: Accepts user input; types include "text", "email", "password", etc.
  • <textarea>: A larger text box for multiline input.
  • <button> or <input type="submit">: Submit button to send the form data.
Learn About Forms on MDN

Comments are added with <!-- comment text --> and are ignored by the browser.
